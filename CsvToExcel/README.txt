1. Скопировать CsvToExcel.exe в рабочий каталог (ваш выбор) и запустить (первый запуск создает файл-настроек)
2. Скопировать в рабочий каталог (пред. шаг) файл данных (*.csv)
3. Открыть каталог %USERPROFILE%\AppData\Local\CSV-to-Excel
4. Открыть файл-настроек csv-to-excel.ini
5. Изменить параметр FILENAME на значение имени файла из шага 2
6. Сохранить файл-настроек csv-to-excel.ini
7. Создать (в каталоге  %USERPROFILE%\AppData\Local\CSV-to-Excel) текстовый файл csv-to-excel.txt с содержимым, которое переносится в Excel
8.Приложение готово к работе
